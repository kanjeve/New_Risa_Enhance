# Risa-Enhancers 

これはVScodeでRisa/Asirのコードを書くための拡張機能 "risa-enhancers"です。

## 機能

「Risa_Enhancers」には、現在次の機能があります。

- コード補完
- シンタックス・ハイライト


## 使い方
VScodeはインストールされているものとします。
VSCode上で`~.rr`の名前でファイルを作り、それに書き込む。
